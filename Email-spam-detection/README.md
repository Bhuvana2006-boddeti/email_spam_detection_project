# 📧 Email Spam Detection

This project detects whether an email message is Spam or Not Spam using Machine Learning.

## 🚀 Features
- Text preprocessing
- TF-IDF Vectorization
- Multinomial Naive Bayes model
- Streamlit Web Application
- 97% Accuracy

## 🛠 Technologies Used
- Python
- Scikit-learn
- Streamlit
- Pandas

## ▶ Run Locally

1. Clone repository
2. Install requirements:
   pip install -r requirements.txt
3. Train model:
   python train_model.py
4. Run app:
   streamlit run app.py

## 📌 Dataset
SMS Spam Collection Dataset
